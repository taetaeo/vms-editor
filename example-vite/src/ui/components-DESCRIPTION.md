# UI 폴더

이 폴더는 프로젝트에서 사용되는 컴포넌트 폴더입니다. 구성은 다음과 같습니다.

| 폴더명   | 설명                                                              | 재사용 |
| -------- | ----------------------------------------------------------------- | :----: |
| atoms/   | 단일 기능의 컴포넌트<br />`e.g. : button, input ...`              |   O    |
| blocks/  | 복합 기능의 컴포넌트<br />`e.g. : inputField, Header, Footer ...` |   O    |
| widgets/ | 컴포넌트내 함수 로직이 있는 컴포넌트                              |   X    |
